h2. IdeasBulb

"IdeasBulb":http://www.ideasbulb.com/ 是为帮助您的公司和自己客户快速沟通交流的Web软件，客户通过IdeasBulb发布自己对公司产品和服务的想法和主意，提供主意的解决方案，评论主意，给方案投票，客户互相交流的同时，也同公司互动，公司可以发现客户的好主意和好建议，将其纳入自己产品和服务中，公司成长了，客户也满意了，双赢。

h2. 版本

版本发布基于Semantic Versioning原则

版本号数字格式如下:

bq. major.minor.patch

构建基于如下的原则:

* major的改变会破坏向后的兼容性，同时重置minor和patch
* minor的改变保证向后的兼容性，同时重置patch
* patch的改变属于修复Bug

了解更多 "Semantic Versioning":http://semver.org/
